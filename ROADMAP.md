## 🐛 Известные проблемы

### Критические
- [ ] Windows: single-worker mode (fork не поддерживается)
- [ ] Android/iOS: только shared library (не standalone)

### Средние
- [ ] Некоторые модули требуют production-like тестирования
- [ ] Документация API в процессе

### Низкие
- [ ] Оптимизация CMake: кэширование, PGO (частично выполнено)
- [ ] Рефакторинг дублирующегося кода

## 📝 Как внести вклад

### Тестирование
1. Склонируйте репозиторий
2. Соберите проект для вашей платформы
3. Запустите тесты: `make test`
4. Сообщите о проблемах через GitHub Issues

### Разработка
1. Создайте fork
2. Создайте ветку: `git checkout -b feature/your-feature`
3. Внесите изменения
4. Запустите тесты
5. Создайте Pull Request

### Документация
1. Найдите неточности в документации
2. Предложите улучшения через PR
3. Добавьте примеры использования
